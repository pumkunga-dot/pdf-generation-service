PDF Generator Service

สร้างใบเสร็จ PDF จาก JSON

Microservice สำหรับรับข้อมูลใบเสร็จ แปลงเป็น PDF และดาวน์โหลดได้ทันที รองรับภาษาไทยและทดสอบผ่าน API ได้จากหน้านี้

สร้างใบเสร็จ

กรอกข้อมูลแล้วกดสร้าง PDF ระบบจะส่ง POST /generate-pdf ให้อัตโนมัติ

API Overview

Endpoints ที่ใช้ในโปรเจกต์นี้

  • POST /generate-pdf
  • GET /pdf/invoice
  • GET /health
